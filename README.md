TCS OHLCV Stock Analysis Dashboard (2025–2026)
An interactive Power BI dashboard analyzing the stock performance of Tata Consultancy Services (TCS) using OHLCV (Open, High, Low, Close, Volume) data for 2025–2026.
The dashboard provides insights into price trends, quarterly and monthly aggregates, trading volume distribution, and key KPIs such as average highs/lows, total open/close values, and volume performance.
Built using Excel (data cleaning & preprocessing) and Power BI (data modeling, DAX measures & visualization) to deliver clear financial insights through interactive slicers and dynamic visuals.
